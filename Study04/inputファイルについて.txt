input.txtファイルはノードとエッジ情報を保存してあるファイルです。
利用するときは
java Dijkstra [出発ノード番号] [目的地ノード番号] < input.txt
の形で実行する必要が有ります。

行番号”i”について”i”の行はノード番号”i-1”のノード情報を表しています。
また各行(各ノード)に接続されているエッジの情報は各行に “[接続先ノード番号]:[エッジコスト],[接続先ノード番号]:[エッジコスト], … [接続先ノード番号]:[エッジコスト]”の形で格納されています。
また最後の行にエッジがなくても’,’だけを記入しているのはそうしないと最終行に相当するノードが生成されないからです。
